动态更新策略

三层触发机制确保 IP 池始终饱满
触发机制 条件 行为
🕐 定时刷新 每 5-10 分钟（可配置） 从所有源拉取最新列表，增量更新
⚠️ 阈值触发 可用 IP 降至最低线以下 立即全量采集 + 降低验证标准临时补充
🔴 紧急补充 可用 IP 归零 跳过验证直接入库（标注 untrusted），边用边验
👤 手动注入 用户通过 CLI/WebUI 添加 直接插入验证流水线，优先级最高

采集器插件接口

每个采集源实现统一接口，可自行扩展
class BaseCollector(ABC):
    """采集器基类 — 实现此接口即可接入自定义源"""

    @property
    def name(self) -> str: ...

    @property
    def interval(self) -> int:  # 采集间隔(秒)

    async def collect(self) -> list[ProxyEntry]:
        """返回采集到的代理列表"""

    async def validate(self, proxy: ProxyEntry) -> bool:
        """验证单个代理是否可用（可选覆盖）"""
