Proxy Vault — 最终设计总览

IP 隐藏代理工具完整设计

项目概览

维度 决定
代理协议HTTP/HTTPS + SOCKS5
IP 来源混合模式：免费池 / 付费API / 自建中继 / Tor — 全部可插拔
CLI 模式守护进程，本地端口代理 (127.0.0.1:1080)
Web UI管理控制台：监控 + 切换模式 + 换IP + 配置 + 启停
中继跳数免费=1跳 / 付费=1-2跳 / 自建=2-5跳 / Tor=内置3跳
单IP模式手动指定外部代理 + 从池中固定一个
可靠性五层防护：预检→心跳→重试→评分→池底保护
免费代理源10+ 源聚合，交叉验证，插件式采集器
动态更新定时刷新 + 阈值触发 + 紧急补充
Tor 集成匿名转发层，可插入链路任意位置
技术栈asyncio / FastAPI / click / aiohttp / python-socks / HTMX / YAML

系统架构（最终版）

架构分层
🖥️ Interface Layer
CLI (click) · Web UI (FastAPI + HTMX) · REST API
🧠 Core Engine
ProxyManager — 代理服务生命周期
RelayChain — 链路编排（free → tor → target）
IPRotator — IP 轮换策略（round-robin / random / weighted）
HealthMonitor — 后台健康检查
ConfigManager — 全局配置管理
🔌 Provider Layer
FreePoolCollector — 免费代理采集验证
PaidAdapter — 付费 API 适配器
SelfRelayManager — 自建中继节点管理
TorRelay — Tor 匿名层（连本地 Tor 守护进程）
CustomProxy — 用户手动指定代理
🌐 Protocol Layer
HTTPServer — aiohttp HTTP 代理
SOCKS5Server — asyncio SOCKS5 代理
ConnectionPool — 连接复用与限流

CLI 命令设计

proxy start    # 启动代理服务（默认：免费池+单跳）
proxy stop     # 停止代理服务
proxy status   # 查看当前状态（模式/IP数/流量）
proxy switch   # 切换代理模式/更换当前IP
proxy config   # 查看/修改配置
proxy web      # 启动 Web 管理控制台

# 启动选项
--provider   free|paid|tor|self|custom   # 代理源
--mode       single|pool                 # 单IP或池模式
--chain      free,tor,self               # 自定义链路
--proxy      socks5://1.2.3.4:1080       # 手动指定代理（single模式）
--port       1080                        # 本地监听端口
--api-key    xxx                         # 付费代理API密钥
