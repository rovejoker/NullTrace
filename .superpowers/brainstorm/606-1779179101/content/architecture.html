模块组成

模块 职责 关键技术
proxy-server 本地 HTTP/SOCKS5 代理服务 asyncio, aiohttp, python-socks
relay-chain 多跳中继链路管理 嵌套 SOCKS5 转发
ip-pool IP 池管理与轮换策略 定时健康检查、加权选择
providers 代理源适配器 插件式加载、统一接口
cli 命令行控制 click
webui Web 管理控制台 FastAPI + 原生 HTML/JS
